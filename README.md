# 🎤 Генератор Рэп-Караоке

Этот скрипт превращает любой рэп-трек в караоке-видео с пословной подсветкой текста. Он автоматически отделяет вокал от минусовки, находит текст песни в интернете, синхронизирует его с аудиодорожкой и создает готовое видео.

## 🌟 Ключевые возможности

- **🎤 Качественное отделение вокала:** Использует современные UVR-модели (`UVR-MDX-NET-Inst_HQ_3`), чтобы аккуратно извлечь основную вокальную дорожку, но оставить бэк-вокал в минусовке для сохранения атмосферы трека.
- **🧠 Умный поиск текста:** Автоматически ищет тексты на **LRCLIB**, **Genius** и других сайтах. Если находит готовый `.lrc` файл с таймингами, использует его для идеальной и мгновенной синхронизации.
- **🤖 Точная синхронизация:** Если готовых таймингов нет, скрипт использует `WhisperX` для привязки найденного текста к аудио (Forced Alignment) или `Whisper` для распознавания с нуля, если текст не был найден.
- **🗣️ Словарь рэп-сленга:** Содержит более 150 специфичных для жанра слов и выражений, которые нейросети часто распознают с ошибками (например, *«оппы»*, *«гуап»*, *«ВВС»*, *«эсщкерит»*). Это значительно повышает качество текста при распознавании "на слух".
- **🚀 Поддержка GPU:** Автоматически определяет и использует видеокарты **NVIDIA (CUDA)** и **AMD (ROCm)** для максимального ускорения всех этапов обработки.
- **⚡ Умное кэширование:** Сохраняет отделенные аудиодорожки и файлы таймингов в `~/.cache/rap_karaoke`. При повторном запуске или изменении параметров рендера не нужно ждать обработки заново — скрипт сразу переходит к созданию видео.
- **🎬 Настраиваемый фон:** Позволяет использовать собственное `.mp4` видео в качестве анимированного фона для караоке.

## 🛠️ Установка

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/your-username/rap_karaoke.git
    cd rap_karaoke
    ```

2.  **Создайте и активируйте виртуальное окружение:**
    ```bash
    python -m venv .venv
    source .venv/bin/activate
    # Для Windows: .venv\Scripts\activate
    ```

3.  **Установите системные зависимости:**
    Вам понадобится `ffmpeg`.
    ```bash
    # Arch Linux
    sudo pacman -S ffmpeg
    # Ubuntu/Debian
    sudo apt update && sudo apt install ffmpeg
    ```

4.  **ВАЖНО: Установите PyTorch:**
    Библиотека PyTorch должна быть установлена **отдельно** до остальных зависимостей, с учетом вашего оборудования. Выберите подходящую команду из `requirements.txt` или на официальном сайте.

    *Пример для NVIDIA с CUDA 12.1:*
    ```bash
    pip install torch torchaudio --index-url https://download.pytorch.org/whl/cu121
    ```
    *Пример для CPU (если нет видеокарты):*
    ```bash
    pip install torch torchaudio --index-url https://download.pytorch.org/whl/cpu
    ```

5.  **Установите остальные зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

## 🚀 Использование

Поместите аудиофайл в папку со скриптом и запустите его, указав имя файла.

#### Простой запуск
Скрипт попытается автоматически определить исполнителя и название из имени файла.
```bash
python rap_karaoke.py "Платина - Санта Клаус.mp3"
```

#### Продвинутые опции
- **Указать исполнителя и название вручную:**
  ```bash
  python rap_karaoke.py track.mp3 -a "PHARAOH" -t "Дико, например"
  ```

- **Использовать фоновое видео:**
  ```bash
  python rap_karaoke.py "Мой Трек.wav" --bg-video background.mp4
  ```

- **Выбрать другую модель Whisper** (влияет на скорость и качество распознавания):
  ```bash
  # Модели: tiny, base, small, medium, large, large-v3
  python rap_karaoke.py track.mp3 --model small
  ```

- **Изменить количество слов в строке:**
  ```bash
  python rap_karaoke.py track.mp3 -w 4
  ```

## ⚙️ Как это работает (пайплайн)

1.  **Хэширование:** Создается MD5-хэш аудиофайла. Если в кэше уже есть обработанные файлы для этого хэша, некоторые шаги будут пропущены.
2.  **Разделение (UVR):** Аудио разделяется на вокальную дорожку (`vocals.wav`) и инструментал (`instrumental.wav`). Результат кэшируется.
3.  **Поиск текста:** Скрипт ищет текст на LRCLIB, Genius и других сайтах.
4.  **Синхронизация (Whisper/WhisperX):**
    -   **Путь А (Идеальный):** Если найден `.lrc` с готовыми таймингами, он используется напрямую.
    -   **Путь Б (Стандартный):** Если найден только текст, `WhisperX` выполняет "forced alignment", чтобы точно сопоставить слова с аудио.
    -   **Путь В (Резервный):** Если текст не найден, `Whisper` распознает его "на слух", а встроенный словарь сленга исправляет потенциальные ошибки.
    -   Результат (файл `.json` с таймингами каждого слова) кэшируется.
5.  **Рендер (MoviePy):** Скрипт генерирует кадр за кадром, рисуя текст поверх фона, и собирает их в финальное видео, подкладывая дорожку с минусом.

## 🎨 Кастомизация

Вы можете легко изменить внешний вид караоке, отредактировав константы в верхней части файла `rap_karaoke.py`:
- `BG_TOP`, `BG_BOT` — цвета градиента фона.
- `COLOR_PAST`, `COLOR_CURRENT`, `COLOR_FUTURE` — цвета для прошедших, текущего и будущих слов.
- `FONT_SIZE_MAIN`, `WORDS_PER_LINE` и другие.
